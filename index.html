最新情報の更新について

本番環境（GitHub Pages）では、画面のボタンから直接データベースを更新する機能は動作しません（静的ホスティングの仕様によるものです）。

学術データベースは、毎日午前6:00と午後18:00（日本時間）にバックグラウンドで自動的に最新論文の検索と更新を行っています。

もし今すぐ手動で最新情報を取得したい場合は、以下の手順を実行してください。

  1. GitHubリポジトリの Actions タブを開きます。
  2. 左側のメニューから Update Academic Feed ワークフローを選択します。
  3. 右側の Run workflow ボタンをクリックして実行します。
